🔐 Authentication

Les routes d'authentification permettent de gérer l'accès à l'API.

POST /api/auth/signup Public

Créer un nouveau compte utilisateur

Body: email, password, firstName, lastName
Response: 201 AuthResponse
POST /api/auth/login Public

Authentifier un utilisateur

Body: email, password
Response: 200 AuthResponse (token, refreshToken)
POST /api/auth/logout ⚠️ Auth Required

Déconnecter l'utilisateur actuel

Response: 200 Success
GET /api/auth/me ⚠️ Auth Required

Récupérer le profil utilisateur actuel

Response: 200 UserResponse
POST /api/auth/forgot-password Public

Demander la réinitialisation du mot de passe

Body: email
Response: 200 MessageResponse
POST /api/auth/reset-password Public

Réinitialiser le mot de passe

Body: token, newPassword
Response: 200 MessageResponse

🌐 Public Routes - Produits

GET /api/public/products Public

Lister tous les produits avec filtres optionnels

Paramètres: page, pageSize, keyword, categoryId, brand, minPrice, maxPrice, minRating, inStockOnly, sortBy
Response: 200 Page<ProductResponse>
GET /api/public/products/{id} Public

Récupérer les détails d'un produit spécifique

Path: id (ID du produit)
Response: 200 ProductResponse | 404 Not Found
GET /api/public/products/{id}/details Public

Détails complets du produit avec produits similaires

Path: id (ID du produit)
Response: 200 ProductDetailWithSimilarsResponse
GET /api/public/products/by-category/{categoryId} Public

Récupérer tous les produits d'une catégorie

Path: categoryId (ID de la catégorie)
Response: 200 Page<ProductResponse>
GET /api/public/products/by-brand/{brand} Public

Récupérer tous les produits d'une marque

Path: brand (Nom de la marque)
Response: 200 Page<ProductResponse>
GET /api/public/products/search/keyword Public

Rechercher des produits par mot-clé

Paramètres: keyword (requis), page, pageSize
Response: 200 Page<ProductResponse>
GET /api/public/products/search/popular Public

Récupérer les produits populaires

Response: 200 Page<ProductResponse>
GET /api/public/products/search/newest Public

Récupérer les nouveaux produits

Response: 200 Page<ProductResponse>

🌐 Public Routes - Catégories

GET /api/public/categories Public

Lister toutes les catégories

Paramètres: page, pageSize
Response: 200 Page<CategoryResponse>
GET /api/public/categories/{id} Public

Récupérer les détails d'une catégorie

Path: id (ID de la catégorie)
Response: 200 CategoryResponse

🌐 Public Routes - Commandes

POST /api/public/orders Public

Créer une commande en tant qu'invité

Body: items, shippingAddress, billingAddress, etc.
Response: 201 OrderResponse
GET /api/public/order-tracking/by-number/{orderNumber} Public

Suivre une commande par son numéro

Path: orderNumber (Numéro de commande)
Response: 200 OrderTrackingResponse
GET /api/public/order-tracking/by-id/{orderId} Public

Suivre une commande par son ID

Path: orderId (ID de la commande)
Response: 200 OrderTrackingResponse

👤 Client Routes

GET /api/client/orders ⚠️ Auth Required

Lister les commandes de l'utilisateur

Paramètres: page, pageSize
Response: 200 Page<OrderResponse>
POST /api/client/orders ⚠️ Auth Required

Créer une nouvelle commande

Body: items, shippingAddress, billingAddress, paymentMethod
Response: 201 OrderResponse
GET /api/client/wishlist ⚠️ Auth Required

Afficher la liste de souhaits

Response: 200 Page<WishlistItemResponse>
POST /api/client/wishlist/{productId} ⚠️ Auth Required

Ajouter un produit à la liste de souhaits

Path: productId (ID du produit)
Response: 201 WishlistItemResponse
DELETE /api/client/wishlist/{productId} ⚠️ Auth Required

Retirer un produit de la liste de souhaits

Response: 200 Success
GET /api/client/loyalty/profile ⚠️ Auth Required

Afficher le profil de fidélité

Response: 200 UserLoyaltyResponse
GET /api/client/notifications ⚠️ Auth Required

Lister les notifications

Paramètres: page, pageSize
Response: 200 Page<NotificationResponse>

👨‍💼 Admin Routes

POST /api/admin/products ⚠️ ADMIN Only

Créer un nouveau produit

Body: name, description, price, stock, category, brand
Response: 201 ProductResponse
PUT /api/admin/products/{id} ⚠️ ADMIN Only

Mettre à jour un produit

Response: 200 ProductResponse
POST /api/admin/categories ⚠️ ADMIN Only

Créer une nouvelle catégorie

Body: name, description
Response: 201 CategoryResponse
GET /api/admin/dashboard/stats ⚠️ ADMIN Only

Récupérer les statistiques du tableau de bord

Response: 200 DashboardStatsResponse
GET /api/admin/orders/{orderId} ⚠️ ADMIN Only

Récupérer les détails d'une commande

Response: 200 OrderResponse
PUT /api/admin/orders/{orderId}/status ⚠️ ADMIN Only

Mettre à jour le statut d'une commande

Paramètres: status (PENDING, PROCESSING, SHIPPED, DELIVERED, CANCELLED)
Response: 200 Success

💰 Finance Routes

POST /api/finance/expenses ⚠️ FINANCE Only

Créer une nouvelle dépense

Body: amount, category, description, date
Response: 201 ExpenseResponse
GET /api/finance/expenses ⚠️ FINANCE Only

Lister toutes les dépenses

Response: 200 List<ExpenseResponse>
GET /api/finance/products ⚠️ FINANCE Only

Lister les produits (vue finance)

Response: 200 List<ProductResponse>

📊 Codes de Réponse HTTP

Code Signification
200 Succès - Requête traitée avec succès
201 Créé - Ressource créée avec succès
400 Mauvaise Requête - Paramètres invalides
401 Non Authentifié - Token required
403 Interdit - Droits insuffisants
404 Non Trouvé - Ressource inexistante
500 Erreur Serveur - Problème interne
